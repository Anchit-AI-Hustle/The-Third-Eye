# Marketing skills — standard toolkit

Reference toolkit to apply on **any marketing-related project** (Kolab, Studio,
the Skills feature, and the Vahdam marketing repos). When building marketing
features, seed these as **Skill templates** (see `lib/skills`) and/or **Kolab
tools** — each has a clear input → output contract.

| # | Skill | Input → Output |
|---|---|---|
| 1 | **Campaign audit** | Paste campaign data → wasted spend + leaks |
| 2 | **Landing page audit** | Paste a URL → headline + CTA fixes |
| 3 | **A/B test analyzer** | Paste results → statistical significance + the next test to run |
| 4 | **Competitor teardown** | Paste a rival's page → angles to beat them |
| 5 | **UTM generator** | Describe the campaign → a clean, consistent tracking spec |
| 6 | **Email sequence** | Give ICP + offer → the full nurture flow |
| 7 | **Content repurposer** | Give one post → a thread, an email, and hooks |
| 8 | **ICP builder** | Paste product + customer quotes → a sharp ICP |
| 9 | **Ad copy matrix** | Give a brief → 20 ad-copy variations |
| 10 | **Creative brief** | Give an angle → a designer-ready brief |

## How to apply
- **Skills feature:** each row is a saved multi-step skill the assistant runs
  (e.g. "Landing page audit" = fetch the URL → analyse headline/CTA/hierarchy →
  return prioritised fixes).
- **Kolab (marketing AI):** expose these as one-click tools with the input form
  on the left and the structured output on the right.
- **Vahdam:** ground outputs in Vahdam's brand/D2C context (the existing
  mailer-architect + lifecycle systems) rather than generic copy.
- Keep the input→output contract above verbatim so the tools stay predictable.
